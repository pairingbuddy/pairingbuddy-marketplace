# Pairing Buddy Marketplace

*Claude Code plugins that accelerate your development workflow without sacrificing quality*

## Quick Start
```bash
/plugin marketplace add pairingbuddy/pairingbuddy-marketplace
```

## Available Plugins

### 🚀 Pairing Buddy

#### Why Pairing Buddy?

Pairing Buddy enforces TDD, SOLID, YAGNI, and Clean Code principles in every response, giving you peace of mind that changes won't break everything.

#### Installation
```bash
/plugin install pairingbuddy-code@pairingbuddy-marketplace
```

**Repository:** [pairingbuddy/pairingbuddy](https://github.com/pairingbuddy/pairingbuddy)
